2. Project Scope

Join is a student group project created for educational purposes. This privacy policy explains how personal data is processed when using the Join web application.

3. Data We Process

3.1 Account and Authentication (Firebase Authentication)

When you register or log in to the application, authentication is handled via Firebase Authentication. During this process, the following personal data is processed and stored:

  • Your name
  • Your email address
  • A technical user identifier (UID) assigned by Firebase

This data is required to create and manage user accounts, to authenticate users, and to associate app content (such as tasks or contacts) with a specific user.

3.2 App Data (Firestore Database)

When you use the application (e.g., creating tasks, contacts, or other entries), the data you provide is stored in Google Firebase Firestore. This may include names, email addresses, phone numbers, task details, and other content you enter within the app.

3.3 Technical Data and Hosting

This application is hosted on infrastructure provided by Developer Akademie GmbH. When you access the website, technical data may be processed automatically, including IP address, date and time of access, requested URL, browser type, and operating system.

This data is processed to ensure the secure and reliable operation of the website and is typically stored in server log files. We do not evaluate this data for marketing purposes.

4. Purposes and Legal Bases

We process personal data for the following purposes:

  • To provide authentication and access control (login/register)

  • To ensure technical functionality, stability, and security

Legal bases may include Art. 6(1)(b) GDPR (performance of a contract / providing the service) and Art. 6(1)(f) GDPR (legitimate interests in operating and securing the service).
